## Font Files

- `AmazonEmber_Rg.ttf` - Regular weight
- `AmazonEmber_Bd.ttf` - Bold weight  
- `AmazonEmber_Lt.ttf` - Light weight

## CDN Usage

These fonts are served via jsDelivr CDN for use in the template customizer:

```css
@font-face {
  font-family: 'Amazon Ember';
  src: url('https://cdn.jsdelivr.net/gh/romadave17/Testfigmamakefreshplaylist@main/fonts/AmazonEmber_Rg.ttf') format('truetype');
  font-weight: 400;
  font-style: normal;
}

@font-face {
  font-family: 'Amazon Ember';
  src: url('https://cdn.jsdelivr.net/gh/romadave17/Testfigmamakefreshplaylist@main/fonts/AmazonEmber_Bd.ttf') format('truetype');
  font-weight: 700;
  font-style: normal;
}

@font-face {
  font-family: 'Amazon Ember';
  src: url('https://cdn.jsdelivr.net/gh/romadave17/Testfigmamakefreshplaylist@main/fonts/AmazonEmber_Lt.ttf') format('truetype');
  font-weight: 300;
  font-style: normal;
}
